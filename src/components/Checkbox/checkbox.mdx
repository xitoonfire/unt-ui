# Checkbox Component Documentation

## Overview

The `Checkbox` component is a simple, customizable checkbox used to select or deselect options. It is commonly used in forms or settings where users can choose from multiple options. This component supports the `disabled` state and is styled with Tailwind CSS.

## Basic Example

```tsx
import React, { useState } from 'react';
import Checkbox from './Checkbox';

function App() {
  const [isChecked, setIsChecked] = useState(false);

  const handleCheckboxChange = (event: React.ChangeEvent<HTMLInputElement>) => {
    setIsChecked(event.target.checked);
  };

  return (
    <div>
      <Checkbox
        id="checkbox1"
        label="Accept terms and conditions"
        checked={isChecked}
        onChange={handleCheckboxChange}
      />
    </div>
  );
}

export default App;
```

## Checkbox Properties

The Checkbox component accepts the following props:

- `id`: The unique identifier for the checkbox element.
- `label`: The text label displayed next to the checkbox.
- `checked`: A boolean indicating whether the checkbox is checked or not.
- `onChange`: A function that handles the change event when the checkbox is toggled.
- `disabled`: A boolean that, when true, disables the checkbox and prevents user interaction.

```tsx
<Checkbox
  id="checkbox1"
  label="Accept terms and conditions"
  checked={isChecked}
  onChange={handleCheckboxChange}
/>
```

## Disabled Checkbox

You can disable the checkbox by passing the `disabled` prop:

```tsx
<Checkbox
  id="checkbox1"
  label="Accept terms and conditions"
  checked={isChecked}
  onChange={handleCheckboxChange}
  disabled
/>
```
