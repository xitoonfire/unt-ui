# UNT UI Component Library Documentation

Welcome to the **UNT UI Component Library**! This library is a collection of reusable, customizable components built using **React**, **TypeScript**, and **Tailwind CSS**. It provides you with a set of high-quality, easy-to-use components to help you build user interfaces quickly and efficiently.

## Key Features

- **Built with React and TypeScript**: Ensures type safety and modularity for your projects.
- **Tailwind CSS**: A utility-first CSS framework to help you rapidly build modern designs.
- **Reusable Components**: A wide variety of pre-built components like buttons, cards, and inputs that you can customize to fit your needs.

## Installation

### 1. Install Tailwind CSS

If you don't have Tailwind CSS installed in your project, you'll need to do so first. Follow the [official Tailwind CSS installation guide](https://tailwindcss.com/docs/installation) to set it up.

### 2. Install the UNT UI Component Library

To install the UNT UI Component Library, run the following command in your project directory:

```bash
npm install unt-ui
```
or

```bash
yarn add unt-ui
```

### 3. Import the Components

Once installed, you can import and use the components in your React project like this:

```tsx
import { Button } from 'unt-ui';
```

Now, you're ready to start using the components in your project!

## Usage

Feel free to explore the components and their documentation to learn how to use them effectively in your applications. Each component has detailed examples and explanations of its available props, customization options, and usage scenarios.

For more information on individual components, check the documentation pages for each one.
